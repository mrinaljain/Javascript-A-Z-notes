# Objects 

An object is a collection of key-value pairs, where each key (property) maps to a value, which can be any data type.

- Dynamic property names

`let user = {
  name:"Prakash",
  age:22,
  college:"abc",
  marks:12,
  ["full name"]:"Prakash Kumar",
  address:{
    city:"Sonpur",
    state:"Bihar"
  }
}`
